# Vefforritun 2 - Hópverkefni 2.

## Keyra verkefni á eigin vél

Gert er ráð fyrir að Node sé nú þegar uppsett.

```bash
git clone git@github.com:emh33/vef2-hopaverkefni2-sse.git
cd vef2-hopaverkefni2-sse
npm install
npm run build
npm run start
```
